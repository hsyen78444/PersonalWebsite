# CS Student Personal Website

Vibe coded a personal website with Codex in 10 minutes.

## Stack

- Next.js App Router
- TypeScript
- Plain CSS
- Component-first structure

## Project structure

```text
app/
  globals.css
  layout.tsx
  page.tsx
components/
  sections/
  ui/
  site-footer.tsx
  site-header.tsx
data/
  site-content.ts
```

## Customize

1. Update your personal details in `data/site-content.ts`.
2. Replace placeholder links for GitHub, LinkedIn, and project demos.
3. Run `npm install`.
4. Start the app with `npm run dev`.
